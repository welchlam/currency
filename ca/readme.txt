## Currency validation description:
0) Currency validation implemented on class CurrencyValidator
1) Currency input should use space as field delimiter
2) Currency input should has 2 fields
3) 1st filed should be 3 upper chars and 2nd filed should be a number
4) Program will show error message for invalid input and continue the program

## Application instruction:
0) jdk 1.8 and mvn 3.3.3 is required.
1) unzip currency.zip.
2) cd ***/currency.
3) mvn clean install exec:java -Dexec.mainClass="com.welch.Main".
4) wait till maven build completed and then the application program will be started, and then you can start to play with it.
